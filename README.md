# 📚 SQL Studies | Jornada de Aprendizado.

Repositório dedicado aos meus estudos em SQL utilizando PostgreSQL como sistema de gerenciamento de banco de dados.

## 🎯 Objetivo

Este repositório tem como finalidade documentar minha evolução prática em SQL, registrando exercícios, projetos e desafios resolvidos ao longo da minha jornada de aprendizado.

O foco principal é:

- 📌 Consolidar fundamentos
- 📌 Praticar modelagem de banco de dados
- 📌 Desenvolver raciocínio lógico aplicado a dados
- 📌 Demonstrar constância e evolução técnica

---

## 🛠 Tecnologias Utilizadas

- PostgreSQL
- SQL (Structured Query Language)
- Git & GitHub

## 📈 Evolução

Este repositório será atualizado constantemente com:

- Novos exercícios
- Desafios práticos
- Modelagens mais complexas
- Consultas avançadas
- Simulações de cenários reais

A proposta é manter uma rotina consistente de prática para evolução contínua.
